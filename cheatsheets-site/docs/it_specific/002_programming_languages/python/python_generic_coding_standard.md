Generic Coding Standard
===============================================================================

File Layout
--------------------------------------------------------------------------------

1.  Shebang or equivalent (i.e. mandatory first line thing)
2.  Header comment
3.  Imports
4.  Init
5.  Constants
6.  Configurable Variables (because of potential Constants Dependencies)
7.  Enums
8.  Classes
9.  Functions
10. Main
11. Close
