Service Accounts
===============================================================================

Naming
-------------------------------------------------------------------------------


Goal is no longer than 15 characters to comply with NetBIOS limitations

| Position | Description                                                                                                                 |
|----------|-----------------------------------------------------------------------------------------------------------------------------|
| 1-3      | *Entity Code* Defines what domain the objcet lives in                                                                       |
| 4        | Delimiter                                                                                                                   |
| 5-11     | *Service Name* - A Friendly name to identify what the service account is for                                                |
| 12       | Delimiter                                                                                                                   |
| 13-15    | *Account Type* - A three digit code that will identify the account type. E.g. SVC for Service, or TSK for scheduled task    |


E.g. `pxy_ldap_svc`
