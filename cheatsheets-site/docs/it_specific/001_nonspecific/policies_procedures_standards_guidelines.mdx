Policies / Procedures / Standards / Guidelines
===============================================================================

Policies
-------------------------------------------------------------------------------

* Compliance is mandatory
* Sets strategic goals by governing organization
* Often compliance-aligned
* Short and to the point
* Clear what the target audience can, cannot, and must do

Standard
-------------------------------------------------------------------------------

* Following is usually voluntary, unless explicitly set otherwise to be mandatory
* Sets normalization of policies and procedures
* Measurable / clearly and unambiguously attainable

Procedure
-------------------------------------------------------------------------------

* Following is mandatory
* Contains step-by-step instructions to accomplish

Guideline
-------------------------------------------------------------------------------

* Following is oluntary
* Contains advice, support, and guidance
* Used to clarify actions when clear direction is not available
